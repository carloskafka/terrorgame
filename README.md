# Jogo de Aventura em HTML5

Um jogo simples de aventura desenvolvido com HTML5, CSS3 e JavaScript. O jogador controla um personagem que deve enfrentar monstros e sobreviver o máximo possível.

## Como Jogar

- Use as teclas **W**, **A**, **S**, **D** para se mover
- Pressione **Espaço** para atacar monstros próximos
- Evite os monstros para não perder vida
- Ganhe pontos ao derrotar monstros
- O jogo aumenta de dificuldade conforme o nível

## Controles

- **W** - Mover para cima
- **A** - Mover para a esquerda
- **S** - Mover para baixo
- **D** - Mover para a direita
- **Espaço** - Atacar

## Recursos

- Sistema de vida e pontuação
- Níveis que aumentam a dificuldade
- Efeitos sonoros
- Partículas visuais
- Interface responsiva

## Tecnologias Utilizadas

- HTML5 Canvas para renderização gráfica
- JavaScript para lógica do jogo
- CSS3 para estilização
- Web Audio API para efeitos sonoros

## Como Executar

Abra o arquivo `joguinho.html` em qualquer navegador moderno.
